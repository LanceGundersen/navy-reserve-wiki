Introduction
============
This project is for complete novices to the readthedocs platform.
If you would like to start documenting with readthedocs, some knowledge of reStructuredText is required.
Here are some resources for beginners to reStructuredText:

#. `Sphinx Primer On reStructuredText <http://www.sphinx-doc.org/en/master/usage/restructuredtext/basics.html>`_ (recommended)
#. `Thomas Cokelaer Cheatsheet <https://thomas-cokelaer.info/tutorials/sphinx/rest_syntax.html>`_
